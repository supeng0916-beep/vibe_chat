## 一、100 字以内产品介绍（两版任选）

**A · 主打「情绪真正驱动匹配」（推荐，约 95 字）**

> VibeChat 用 AI 把你此刻的一段心情解析成「情绪坐标」（效价 × 唤醒度），再按情绪相似度把你和同频的陌生人匹配进匿名对话。情绪不只是标签，而是真正驱动匹配的数学依据，并在「情绪星图」上可见。支持 OpenAI、Anthropic 双标准，外部异常自动降级，未匹配时由 AI 同频陪伴兜底。

**B · 更短更口语（约 70 字）**

> 写下此刻的心情，VibeChat 用 AI 把它变成情绪坐标，帮你找到同频的陌生人匿名聊天。情绪在「星图」上看得见、真的决定匹配；OpenAI／Anthropic 双标准随心切换，永远有人（或 AI）懂你此刻。

## 二、演示视频脚本（3–6 分钟，逐镜）

> 录制前准备：① 先打开 `https://vibechat-api-jx7z.onrender.com/api/health` 预热后端（免费档会休眠）；② 准备**两个浏览器窗口/无痕窗口**用于演示双人匹配；③ 关掉多余标签，全屏。

| 时间 | 画面（拍什么） | 解说（说什么 / 字幕） |
|---|---|---|
| 0:00–0:20 | 打开 https://vibe-chat-psi.vercel.app 落地页，星空主题、标题动画 | 「这是 VibeChat——一款 AI 驱动的情绪社交应用。它先读懂你此刻的情绪，再把你匹配给同频的人。」 |
| 0:20–1:00 | 在输入框写一段真实心情（如"面试前好紧张又有点期待"），点「生成情绪坐标」，跳到情绪结果页 | 「我写下心情，AI 不只给标签，而是解析出效价、唤醒度、强度等结构化『情绪坐标』。」 |
| 1:00–1:40 | 镜头停在**情绪星图**：发光的星落在象限里 + 关键词 + 情绪色 | 「这是核心——它把情绪映射成二维坐标里的一颗星。效价决定左右，唤醒度决定上下。这张坐标，就是后面匹配的依据。」 |
| 1:40–2:30 | 点「寻找同频的人」。**窗口①**先进入匹配等待（星图候选漂浮）；切到**窗口②**用相近心情也走到匹配 → 两边瞬间匹配进同一房间 | 「匹配不是随机聊天室——系统用两人情绪坐标的加权距离找最近邻。另一个用户情绪相近，立刻被连到一起。」 |
| 2:30–3:30 | 在两个窗口间真实对话：发消息、对方实时收到、对方情绪徽章、点「AI 破冰」生成开场白 | 「真实的实时匿名聊天，双方身份由系统生成。AI 还能根据双方情绪生成破冰开场白，让对话自然开始。」 |
| 3:30–4:10 | 新开一个窗口，单人走到匹配 → 等约 12 秒 → 进入「同频陪伴」房间，和 AI 聊几句 | 「如果此刻没有同频的人在线，也不会冷场——AI 会扮演一个情绪相近的匿名网友陪你聊，这是兜底机制。」 |
| 4:10–4:45 | 顶栏徽章「OpenAI 标准 · gpt-5.5」；打开代码 `backend/app/llm/`（openai_provider / anthropic_provider / factory），讲一句切换 | 「技术上，OpenAI 与 Anthropic 两套标准接口都做了原生适配，一个环境变量即可切换；外部 API 异常会自动降级到本地分析，演示永不中断。」 |
| 4:45–5:10 | 输入一段极端负面文字，展示聊天页顶部**温和求助提示条** | 「识别到极端负面情绪时，产品会给出温和的求助提示——AI 情绪社交也要有安全边界。」 |
| 5:10–5:40 | 回到落地页/星图收尾，打字幕列出：FastAPI＋Next.js、情绪向量匹配、双 LLM 标准、已上线 Vercel＋Render | 「VibeChat：让情绪真正成为社交的起点。前后端分离、已部署上线，欢迎体验。」 |

**录制要点**
- 一定要拍出「**情绪不同 → 匹配/房间不同**」的对比，正面回应评审最看重的"情绪是否真驱动匹配"。
- 双人匹配用两个无痕窗口，分别写**相近**心情即可秒配；写**差异很大**的心情则会各自走到 AI 兜底，可顺势演示兜底。
- 双标准如不想在镜头前等 Render 重部署，用"顶栏徽章 + 代码结构"来证明即可。
